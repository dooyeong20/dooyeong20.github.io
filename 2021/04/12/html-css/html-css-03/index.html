선택자의 역할

HTML에 스타일(CSS)을 적용하기 위해 HTML의 특정한 요소를 선택하는 사인(sign)

  • h1 태그에 색을 빨간색으로
  • p 태그의 색을 파란색으로

속성(Properties)과 값(Value)

  • HTML의 속성은 attribute
  • CSS 속성은 properties
