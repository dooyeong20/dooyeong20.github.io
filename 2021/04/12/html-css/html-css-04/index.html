HTML 주요 범위와 메타데이터

블록 요소

  • DIV, H1, P 등
  • 사용가능한 최대 가로 너비를 사용
  • 가로 사이즈를 지정하지 않고 height만 지정해준다면 가로사이즈를 끝까지 채움
  • width: 100%; height: 0;으로 시작 (auto)
  • 수직으로 쌓임
  • margin, padding 위,아래,좌,우 모두 사용 가능
  • For Layout

인라인 요소

  • SPAN, IMG

  • 필요한 만큼의 너비만 사용

  • 크기를 지정할 수 없음

  • width: 0; height: 0;으로 시작 (auto)

  • 수평으로 쌓임

  • 태그 사이에 띄어쓰기가 페이지에 그대로 보인다.

  • margin, padding 위,아래를 사용할 수 없다.

  • For Text

인라인, 블록 요소 변경

display속성을 활용해 변경할 수 있다.
