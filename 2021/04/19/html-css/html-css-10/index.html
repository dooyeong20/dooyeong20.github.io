HTML

1993년 최초버전. W3C가 XHTML1.0을 구체화하기 위해 XHTML2.0작업을 진행을 시도

  • HTML 4.01 vs XHTML 1.0

    • 재정의 한 것이기 때문에 비슷함
    • HTML 은 자유도가 너무 높았음
    • 애플, 모질라, 오페라가 w3c와 별개로 작업함(WHATWG)

  • HTML5 정보 구조를 명확히 할 수 있도록 아웃라인 알고리즘(?)이라는 것이 있음

  • HTML5는 HTML 4.01과 XHTML1.0을 모두 수용했다.
