104. Maximum Depth of Binary Tree

  • 여기서 문제를 풀어볼 수 있습니다.

문제 설명

Given the root of a binary tree, return its maximum depth.

A binary tree’s maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

문제 풀이

leetcode 문제가 질이 좋다는 얘기를 듣고 leetcode의 문제도 함께 풀어보고자 하여 풀게된 문제이다. 해당 문제는 이진 트리에서 간단하게 depth를 구하는 문제이다. 기본적인 dfs가 가능하다면 쉽게 풀 수 있는 문제로, 각각의 재귀 호출시마다 현재 저장하고 있는 최대 깊이(tmp)와 현재 깊이를 비교해주면서 최댓값을 리턴해주면 된다.
