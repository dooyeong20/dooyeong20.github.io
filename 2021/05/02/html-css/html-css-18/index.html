VLIVE 팀 프로젝트

VLIVE 반응형 프로토타이핑

VLIVE는 반응형 레이아웃 배치 breakpoint가 크게 총 2개이다. 모바일, 태블릿, 데스크탑 모드 총 3개의 반응형 레이아웃을 제공하고 있는 것을 보았고, 지난 글에서 볼 수 있듯이 헤더, 좌/우측 컨테이너 그리고 메인(동영상)컨테이너, 그리고 댓글 컨테이너로 구성되어 있는 것을 확인했다.

따라서 이대로 레이아웃을 짜보았는데, VLIVE는 원래 flex 로 배치되어 있었지만 나는 grid 를 사용해 배치해보았다. 왜냐하면 큰 단위의 전체적인 배치를 하고, 반응형에 대응하기에는 grid 가 적합하다고 판단했기 때문이다.

데스크탑 레이아웃

데스크탑 레이아웃은 헤더와 좌우측 컨테이너, 그리고 메인과 댓글 창의 배치가 위와 같이 되어있다.

태블릿 레이아웃

태블릿 레이아웃은 우측 컨테이너가 메인 컨테이너와 댓글 컨테이너 사이로 배치되는 것을 확인할 수 있는데, 이 때문에 flex 배치보다 grid 배치를 선택하게 되었다.

모바일 레이아웃

모바일 레이아웃은 위 태블릿 레이아웃에서 좌측 컨테이너가 토글로 숨어있는 형태로 구성된다.
