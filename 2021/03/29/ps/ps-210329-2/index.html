Problem Solving - 트리의 부모 찾기(S2)

1. 문제 [ Link ]

루트 없는 트리가 주어진다. 이때, 트리의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.

2. 입력

첫째 줄에 노드의 개수 N (2 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N-1개의 줄에 트리 상에서 연결된 두 정점이 주어진다.

3. 예제 입/출력

4. 풀이

트리의 연결 정보와 루트 노드가 1이라는 정보만으로 트리의 구조를 파악해 각 노드들의 부모를 찾아 출력하는 문제이다.

문제를 풀이하기 위해 dfs를 사용하여 루트노드(1) 부터 시작해 각 노드들을 탐색해주며, 그 과정에서 각각의 부모를 설정해주는 방식으로 풀이했다.

그리고 파이썬의 재귀함수 호출 제한으로 인해 런타임 에러가 났었기 때문에, sys의 setrecursionlimit()을 이용해 재귀호출 limit을 늘려주었다.
